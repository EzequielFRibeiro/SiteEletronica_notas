entao!# Assistec Eletrônica

Sistema completo para assistência técnica de eletrônica.

## Funcionalidades
- Cadastro e listagem de clientes
- Entrada e saída de notas
- Controle de aparelhos (em análise, prontos, entregues)
- Informações de contato
- Importação automática dos clientes do backup JSON

## Como rodar
1. Instale as dependências do backend e frontend
2. Execute o backend
3. Execute o frontend

## Estrutura
- `backend/` - API Node.js + Express + SQLite
- `frontend/` - Interface moderna baseada no HTML fornecido
- `database/` - Arquivos do banco de dados
- `backup_os_2025-06-28.json` - Backup dos clientes
